# 项目缺失功能补全计划

经过对比《项目文档》，目前系统核心骨架已搭建，但仍存在以下功能缺失：

## 1. 核心业务流程 (优先级：高)
- [ ] **录制模式切换**：前端 `Recorder.vue` 缺少“单人录制”与“多人对话”的模式切换及对应引导。
- [ ] **标注流程闭环**：

- [ ] **榜单计算机制**：后端缺少定时任务（Cron Job）来定期统计时长并更新 `fangyan_rankings` 表。

- [ ] **系统设置**：`Settings.vue` 目前为空，需实现系统参数配置。

- [ ] **数据库迁移**：`database/migrations` 目录为空，缺少版本化管理。
- [ ] **运维脚本**：`scripts/` 目录为空，缺少部署脚本。
