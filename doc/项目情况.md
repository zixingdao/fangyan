# 项目情况

## 1. 项目概述

*   **项目背景**：采集学生的长沙方言，给阿里公司的项目。
*   **流程**：
    1.  拉人进钉钉群
    2.  试音、上传试音成功证明
    3.  拉进正式群
    4.  按流程录音并上传到群里
    5.  后期进行数据标注
*   **目标**：总时长达到 5000 小时以上。

## 2. 规则

1.  **单人**：最多 20 小时。
2.  **多人**：最多不超过 100 小时。

## 3. 宣传方式


1.  网站宣传
2.  荣誉激励：
    *   长沙大使之类的荣誉
    *   奖金
    *   证书

## 4. 管理法

### 辅助人进行方言的讲述

*   **单人方言采集**：制作引导型文档，启发录音者自己想说的话题。
*   **多人方言采集**：例如与父母的对话，提供一个话题，引导父母和孩子进行和谐的沟通。
