# 长沙方言采集平台 (Changsha Dialect Collection Platform)

## 项目简介
"湘音传承·长沙方言守护计划" - 为阿里公司采集长沙方言数据，目标总时长达到5000小时以上。

## 目录结构
- `frontend/`: 前端项目 (Vue 3 + TypeScript + Vite)
- `backend/`: 后端项目 (Node.js + Express + TypeScript)
- `database/`: 数据库初始化脚本
- `doc/`: 项目文档

## 快速开始

### 前端启动
```bash
cd frontend
npm install
npm run dev
```

### 后端启动
```bash
cd backend
npm install
npm run dev
```
